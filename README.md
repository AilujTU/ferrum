# Ferrum

## Table of Contents
- [Installation Guide](#installation-guide)
- [Used Resources](#used-resources)

This is a simple Chromium browser extension to help keep you focused on what you want to achieve and restrict your access to distracting sites, such as social media sites.

Currently it supports selecting a timeframe where selected websites are blocked, including or excluding the weekend. Websites are added or removed in the extension settings. It also keeps track of previously added sites for ease of adding them back on the restricted list.

If you're interested to see what features are added next, head to the issue section of this github. 
If you want to use the extension yourself, simply read on to find the installation guide.


## Installation Guide
> Prequistite: Installed a Chromium browser, such as Chrome, Edge, etc.

This installation guide will focus on the installation in Chrome, other installation processes may vary slightly, but should generally follow the same pattern.

1. Scroll up this GitHub page, then click on the green `Code` button and then on `Download ZIP`.
    ![A zoomed in view of this GitHub page, the green 'Code' button is circled in red, next to it is a red 1. It should be clicked first. The 'Download ZIP' button is circled in red, next to it is a red 2. It should be clicked second.](/assets/install_guide_pt0.png)
2. Extract the .zip folder.
2. Click on the puzzle symbol on the right side of the navigation bar.
    ![A zoomed in view of a opened Chrome tab, the puzzle symbol in the right corner is circled in red. It should be clicked.](/assets/install_guide_pt1.png)
3. Navigate to the extension manager screen with a click on `Manage Extensions`, the last option in the popup screen opened previously.
    ![A zoomed in view of a popup menu, the option 'Manage Extensions' is circled in red. It should be clicked.](/assets/install_guide_pt2.png)
4. Activate developer mode in the upper right corner of the extension manager screen.
    ![A view of the extension manager screen, in the upper right corner the developer mode option is circled in red. It should be activated.](/assets/install_guide_pt3.png)
5. Click on `Load unpacked` and select the previously downloaded & extracted `ferrum-main` folder.
    ![A view of the extension manager screen, in the upper left corner the 'Load unpacked' button is circled in red. It should be clicked.](/assets/install_guide_pt4.png)
6. And voila! The extension is successfully loaded and ready for use.
